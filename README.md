## Run Locally

Clone the project

```bash
  git clone https://github.com/naimuddin94/MK_shop_client.git
```

Go to the project directory

```bash
  cd MK_shop_client
```

Set environment variables fllowing env.exaple file

Install dependencies

```bash
  npm install
```

Start the server development

```bash
  npm run dev
```

Build project

```bash
  npm run build
```
